# MDX Components

The theme supports component blocks in markdown.

## Syntax

Use:

:::component <Name>
key: "value"
key2: "value"
:::

## Supported components

### Hero
Keys:
- headline (string)
- subheadline (string)
- cta_text (string)
- cta_href (string)
- secondary_text (string)
- secondary_href (string)

### YouTubeVideo
Keys:
- video_id (string)  e.g. "dQw4w9WgXcQ"
- title (string)
- caption (string)
